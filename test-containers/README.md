# README.md

This demo Spring Application is used as an example for testcontainers

## Requirements

- Java 21
- Docker

## Spring Dependencies

- Spring Web
- Spring Data JPA
- PostgreSQL Driver
- Testcontainers

## How to run

In order to run the testcontainers run the following

```sh
./gradlew test
```
